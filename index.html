A tag p é utilizada para definir parágrafos.

Para alinhar eu no título fiz "h2 align="center" porque h é de head/títlo se eu quiser alinhar um parágrafo uso "p align="center"

A tag b é negrito

A tag i é itálico

A tag u é sublinhar

A tag hr é para criar linhas (literalmente)

Para fazer o efeito H2O usa-se a tag sub que faz o número ou letra ir para baixo, para mandar para cima é a tag sup exemplo: 10-6
