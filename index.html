As tags de HTML:

A tag p é utilizada para definir parágrafos.

Para alinhar eu no título fiz "h2 align="center" porque h é de head/títlo se eu quiser alinhar um parágrafo uso "p align="center"

A tag b é negrito

A tag i é itálico

A tag u é sublinhar

A tag hr é para criar linhas (literalmente) e para mudar o comprimento da linha usa-se "hr width="15%"

Para fazer o efeito H2O usa-se a tag sub que faz o número ou letra ir para baixo, para mandar para cima é a tag sup exemplo: 10-6

  • Exemplo de parágrafo com tipo de letra Arial tamanho 5 cor vermelho

  • Exemplo de parágrafo com tipo de letra Georgia tamanho 6 cor verde escuro

Para trocar a a fonte usa-se "font face="Arial""e para trocar a cor e tamanho usa-se "font size="5"" e para trocar a cor usa-se "color="red""

Para fazer uma lista usa-se o ul e li exemplo: ""ul" "li" coisa 1 "li" coisa 2 "/ul" o li e ul são listagens não ordenadas

Lista não ordenada

  • Sapo
  • Cão
  • Gato

Listas não ordenadas podem usar o símbolo redondo ou quadrado usando "ul type="square"

  • Raposa
  • Rato
  • Raia

Lista ordenada

Para fazer uma lista ordenada usa-se o "ol"

  1. Água
  2. Gelo
  3. Vapor

Para as listas ordenadas podemos usar também letras, nºs romanos ou traços

  1. Café
  2. Chá
  3. Leite

Dá para fazer uma listagem ordenada começar por um nº à noss escolha também ao usar "ol start="7""

  1. Fogo
  2. Água
  3. Pedra
  4. Vento
